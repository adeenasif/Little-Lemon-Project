# Little-Lemon-Project

## Little Lemon Restaurant - Booking System

A modern, user-friendly reservation system for the Little Lemon restaurant, developed using React.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop screens
- **Accessibility**:Meets WCAG 2.1 AA standards with appropriate ARIA attributes
- **Form Validation**: Client-side validation with meaningful error messages
- **Unit Tests**: Comprehensive test coverage using Jest and React Testing Library
- **State Management**: Context API for global state management
- **Routing**: React Router for navigation
- **Semantic HTML**: Clean and meaningful use of semantic HTML elements
- **Error Handling**: User-friendly error handling with informative feedback

## Prerequisites

- Node.js 14.0 or higher
- npm or yarn
- Git
